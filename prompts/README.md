# Spec-first prompt pack

These prompts route future work through the repository's local contracts. For
behavioral work, always read `AGENTS.md`, `docs/spec-first-workflow.md`,
`docs/specs/README.md`, and `docs/specs/index.md` before implementation source.

- [`day-to-day-spec-first.md`](day-to-day-spec-first.md) — normal feature, bug,
  refactor, and behavior-investigation work.
- [`brownfield-discovery.md`](brownfield-discovery.md) — create or repair
  first-pass specs without implementation changes.
- [`generate-first-specs.md`](generate-first-specs.md) — turn a bounded
  discovery result into product contracts.
- [`spec-first-workflow-repair.md`](spec-first-workflow-repair.md) — audit a
  workflow migration without product implementation changes.
